# Selenium Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## Project Structure
- src/test/java → Test cases
- pom.xml → Dependencies
- testng.xml → Test execution

## Features
- Multiple test scenarios
- Basic automation framework
- TestNG integration

## How to Run
1. Clone the repository
2. Open in Eclipse
3. Run testng.xml
